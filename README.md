# TweetOnClick-Chrome-Extension
Tweet what you want on just a click 

## Built With

* Javascript
* JQuery

## Installation

Windows & Linux:

```sh
Open google chrome (or Opera/firefox)
```

```sh
Go to More tools -> extensions
```

```sh
add unpacked extensions
```

```sh
start using :)
```
